Freedeck + Spotify

Spotify is not affiliated with Freedeck, any reference to the specific name "Spotify" is towards the Spotify plugin.

Here's how to setup your integration.

Why do I need to do this? Spotify recently pushed new requirements (250k+ active users per month) which doesn't allow Freedeck to become a large Spotify API app.

Step 1

Log into the Spotify Developer Dashboard (click here or go to https://developer.spotify.com/dashboard)

Once logged in, you should see this dashboard:

Step 2

Create a new app (click here or go to https://developer.spotify.com/dashboard/create)

Here, you can input any name and description.

It is very important that you set the Redirect URI to (click to copy) and enable the Web API and Web Playback SDK.

Once you type the Redirect URI, press Add.

Once you've inputted all the data, press Save.

Step 3

Now, input your app's Client ID. It will be saved to the Spotify plugin settings.

You can click the circled clipboard icon in the dashboard to copy it to your clipboard.


Step 4

If you see a pop up window from Spotify, hit Authorize. Freedeck will now take care of the rest!
